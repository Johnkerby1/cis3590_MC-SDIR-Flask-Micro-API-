# Flask Micro-API – CIS 3590

## Overview

This project is a minimal Flask web API developed for the CIS 3590 Flask Micro-API assignment.  
The application demonstrates basic RESTful API design, JSON responses, and GET/POST functionality.

The API includes two main endpoints:

- `/hello` – Returns a welcome message in JSON format.
- `/data` – Returns sample structured data and allows new data to be added via POST.

---

## Technologies Used

- Python 3.x
- Flask
- Postman (for API testing)
